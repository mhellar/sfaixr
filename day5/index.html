• a-entity tag
  • So far we've mainly been working with the built in aframe primitive like a-box, a-sphere etc..
  • Theres another way to add elements to the scene with a-entity
  • By itself, a-entity has no appearance, behavior, or functionality.
  • To extend a-entity you attach a component to it which gives it new atrributes
  • There are many avialable components which are built into aframe and externally
  • Lets look at the built in a-frame ones https://aframe.io/docs/0.8.0/components/background.html
  • Components are written in Javascript
Components of A-Frame’s entity-component framework are JavaScript modules that can be mixed, matched, and composed onto entities to build appearance, behavior, and functionality. We can register component in JavaScript and and add it to an a-entity tag. Components are configurable, reusable, and shareable. Most code in an A-Frame application should live within components.
